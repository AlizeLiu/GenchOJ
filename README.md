## 简述
+ 轻量级，易于部署和自定义定制
+ 前后端分离，提高服务器性能
+ 支持多机器多进程判题，判题更高效
+ 支持 C/C++/Java/Python2/Python3和Swift5.1语言
+ 支持 Special Judge和选择题判题
+ 丰富的API，开放的源代码
+ 一键保存和导出代码模板
+ **支持类似LeetCode的模板题判题功能**
+ **新增班级管理与布置作业功能**
